IT Project

Thermal sensor for temperature screenings. The idea is to build a simple thermal scanner that reads peoples temperatures as they walk into a public place such as a retail store or movie cinema. This will offer many businesses a low cost way to provide temperature screenings for patrons/customers in the current pandemic climate. To keep it low cost we will be using a Raspberry Pi4 and an Adafuit IR Thermal camera, which is approximately $150.00 in cost. To interpret the data, Python code will be used and run on the Raspberry Pi.
